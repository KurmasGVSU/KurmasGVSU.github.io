Pandas Introduction

  • import pandas as pd
    • The as part allows you to say pd instead of the “full” pandas
  • Open a .csv or .xlsx.
    • File can be compressed. No need to do anything special (as long as system can determine what kind of compression)
  • DataFrame is a 2D representation of data.
    • Kind of like a worksheet
  • Accessing like an array gives a column by name
    • print(df['name'])
    • print(df['team'])
    • print(df[['name', 'team']])
      • Notice the use of the nested array [[
  • To access rows, use loc or iloc
    • iloc uses row numbers
    • loc uses row index.
      • Need to specify index column
      • can be done with index_col param to read_*
  • Can specify multiple rows:
    • df.loc[['WAS', 'JAX']]
    • Again note the nested array.
    • Second parameter is a list of desired columns.

Pandas Filters

  • Applying a logical operator to a column gives a series of True and False.
    • df['team'] == 'JAX'
  • Passing this result to df[] generates the values that pass the filter.
    • df[df['team'] == 'JAX']
  • See a given column for rows matching a criteria.
    • print(df.loc[df['team'] == 'JAX', 'games'])
