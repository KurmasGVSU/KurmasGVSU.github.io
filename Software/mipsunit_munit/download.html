Usage

Compile: Simply put munit.jar in your classpath to compile your test clas: javac -cp munit.jar InRangeTest.java.

Run: Run munit.jar and pass it (1) the name of the assembly file under test, and (2) the name of the class file containing the tests: javac -jar munit.jar in_range_correct.asm InRangeTest.class
