CIS 351

Combinatorial Circuit with Choices

Winter 2021
  1. Design a circuit that increments the hours and minutes on a 24-hour clock. This circuit has a 5-bit input representing hours and a 6 bit input representing minutes. It also has a five-bit output representing the updated hour and a 6-bit output representing the updated minute. The outputs should represent one minute after the inputs. For example, if the inputs are 6 and 10, then the outputs should be 6 and 11. If the inputs are 7 and 59, then the outputs should be 8 and 0. If the inputs are 23 and 59, then the outputs should be 0 and 0.
  2. Design an 4-to-2 encoder
  3. Design an 8-to-3 encoder.
    • How would you build it from "scratch"?
    • How would you build it using two 4-to-2 encoders?
  4. Design a circuit that takes 3 integers as input and returns the largest of the three. Assume you have a "greater than" subcircuit. Also assume that all three inputs are unique.
  5. Design a circuit that performs signed multiplication. Assume you already have a circuit that performs unsigned multiplication. Then add gates before and after this circuit to correctly handle the sign.
