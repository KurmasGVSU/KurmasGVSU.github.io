CIS 263

Intro to C++

Fall 2019

The objective of this assignment is simply to help you get started using git and C++ in the GVSU environment.

Write a simple C++ class and "driver" program. The class must

  • Have at least two private instance variables.
  • Have at least three methods.

The driver program must:

  • Instantiate at least two instances of the class.
  • Use cout.

Finally, you must also include either:

  • A shell script that builds your executable in EOS, or
  • A makefile that will run in EOS.

Submission and grading

You will use Github Classroom to submit this assignment.

  • If you don't already have a github account, create one.
  • Visiting classroom.github.com/a/GIouGesI. This will create a repository named intro-to-cpp-username where username is your GitHub account name.
  • Clone this new repository, complete your assignment, then commit the changes.
  • Make sure your last commit message indicates that the assignment is ready for grading.

Remember: I will be using EOS to grade your assignments. Make sure your code compiles and runs there.
